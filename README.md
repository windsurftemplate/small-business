# Professional Business Website Template

A modern, responsive business website template built with HTML5, CSS3, and JavaScript. Features a sleek design with smooth animations and professional styling.

## Features

- **Modern Hero Section**
  - Dynamic SVG background with subtle animations
  - Responsive design with optimal text contrast
  - Clear call-to-action buttons
  - Professional typography and spacing

- **Navigation**
  - Clean, responsive navigation bar
  - Authentication buttons (Login/Sign Up)
  - Smooth scroll behavior
  - Mobile-friendly design

- **About Page**
  - Company story and mission
  - Team member profiles
  - Core values section
  - Company statistics

- **Testimonials**
  - Client feedback showcase
  - Professional testimonial cards
  - Avatar integration
  - Responsive grid layout

## Technical Details

### File Structure
```
small-business/
├── index.html          # Main landing page
├── about.html          # About page
├── css/
│   └── style.css      # Main stylesheet
├── js/
│   └── main.js        # JavaScript functionality
└── images/
    ├── hero-bg-new.svg    # Hero background
    ├── about-main.svg     # About page main image
    ├── team-*.svg         # Team member avatars
    └── testimonial-*.svg  # Testimonial avatars
```

### Technologies Used
- HTML5

- SVG Animations

- Responsive Design

- Chrome (latest)

   - Update text in `index.html` and `about.html`
   - Modify colors in `css/style.css`
   - Adjust animations in `js/main.js`

## Customization

### Colors
The main color scheme is defined in CSS variables:
- Primary: Deep Navy (#1a1c2e)
- Secondary: Forest Green (#2d614c)
- Accent: Golden Yellow (#FFD700)
- Text: White (#ffffff)

### Typography
- Headings: Poppins
- Body: Playfair Display
- Custom font sizes for different screen sizes
